# Phishing-Email-Threat-Detector
AI-Powered Email Security assistent that detects phishing emails ,analyzes risk and provides actionable security insights.

An AI-powered cybersecurity tool designed to detect phishing emails and protect users from online threats.

This project analyzes email content using Machine Learning and identifies whether an email is safe or malicious. It also provides a risk score, explains why the email is flagged, and suggests safety actions to the user.

## 🚀 Features
- Phishing Email Detection (ML-based)
- Risk Level Analysis (Low / Medium / High)
- Suspicious Keyword & Link Detection
- Explainable Output (Why it's phishing)
- Security Advice for Users
- Batch Email Analysis (CSV support)

## 🧠 Technologies Used
- Python
- Scikit-learn
- Pandas
- Natural Language Processing (TF-IDF)

## 🎯 Purpose
To help individuals and organizations identify phishing attacks and improve cybersecurity awareness.
